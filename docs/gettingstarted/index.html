Getting Started

Install

  1. Sidewinder uses Java 1.8+
  2. Download latest jar from Releases page
  3. Create an empty sidewinder.yaml file
  4. Run using java -jar jar_name_here server sidewinder.yaml to launch with defaults
Please refer to the elaborate installation instructions and options on the Downloads page.

Grafana

  1. Download Grafana from https://grafana.com/
  2. Download Sidewinder Grafana Data Source from Releases page
  3. Unzip and copy to Grafana plugins directory
  4. Start Grafana
  5. Navigate to Grafana UI
  6. Configure Sidewinder data source (name "_internal") in Grafana -> Data Sources -> Add data source -> Type=Sidewinder
  7. Provide URL of Sidewinder http://localhost:8080/_internal
  8. Save and Test data source
  9. Create new Dashboard -> Graph and select "_internal"
